Deaf Sign Language Prediction

Developed a CNN-based system to interpret sign language from images. Enhanced accessibility for the deaf community by converting gestures to text.

The Deaf Sign Language System is an intelligent communication tool designed to bridge the gap between the hearing-impaired and the hearing world. Utilizing computer vision and machine learning techniques, the system interprets hand gestures associated with sign language and converts them into readable or audible text in real-time. By capturing hand movements through a camera and processing them using trained models, this system enables effective two-way communication for deaf or hard-of-hearing individuals. Key features include: Real-time gesture recognition Conversion of sign language to text or speech User-friendly interface for both hearing and non-hearing users Support for commonly used sign languages

  • Tech: Python, CNN, OpenCV, Scikit-learn
  • Output: Real-time gesture recognition
